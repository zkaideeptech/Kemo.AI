# UI Spec (SaaS)

## Visual direction
- Mature SaaS look, not "AI demo"
- Clean surfaces, subtle borders, confident typography
- Avoid novelty gradients or gimmicks

## Typography
- Headings: 20-28px, semi-bold
- Body: 14-16px, regular
- Helper text: 12-13px, muted

## Spacing
- Page padding: 24px desktop, 16px mobile
- Section gap: 24-32px
- Card padding: 20-24px
- Form field spacing: 12-16px

## Buttons
- Primary: solid background, high contrast
- Secondary: outline
- Tertiary: ghost
- Destructive: red emphasis

## Tables
- Minimal borders, strong row hover
- Align numbers right
- Status badges visible

## Empty states
- Title + short explanation + primary CTA
- Avoid overlong copy

## Loading
- Use skeletons for lists and cards
- Use spinner for single action

## Error states
- Short, direct message
- Provide next action (retry/support)

## Language switch
- Visible in header (top right)
- Persist selection

## Plan gating UI
- Free plan must show 50MB single-file limit
- Show “Supabase Storage is project-level” note
- Offer upgrade hint when limit exceeded
